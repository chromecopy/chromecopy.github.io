Overveiw

Copy and paste more stuff with Copy Master. Copy Master can copy and paste text and save it in 9 slots. Currently not avalible on the toolbar.

Permissions

Copy Master need the permission: 'declarativeContent' to get access to websites and permission: 'storage' to store the clipboard to remember the clipboard not only on the active tab but also to store it to other tabs.

  • declarativeContent
  • storage

Shortcuts

Copy: Alt + 1-9
Paste: Alt + Shift + 1-9

Bookmark

Installation

1. Download js, zip, 7z, or the text.

2. Create a bookmark (www.computerhope.com) and paste the code in the url field to activate it.

How to use

When finished with the installation, click on in the bookmark field.

Disadvantage: You will have to use the bookmark on each website.
